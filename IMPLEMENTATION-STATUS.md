## Quick Reference

### Service-Location URL Naming Convention
Pattern: `/[service]-[location].html`

**Examples:**
- `/web-design-midrand.html`
- `/logo-design-johannesburg.html`
- `/printing-sandton.html`

### Location Hub URL Pattern
Pattern: `/services-[location].html`

**Examples:**
- `/services-midrand.html`
- `/services-waterfall.html`

### Main Service URL Pattern
Pattern: `/ourservices/[service].html`

**Examples:**
- `/ourservices/web-development.html`
- `/ourservices/brandinganddesign.html`
